
# TWC Contact Portal Client-side App


## Tech Stack

ReactJS, Vite, TailwindCSS
## Run Locally


```bash
  https://github.com/ashandimantha2000/twc-test-web.git
```

Go to the project directory

```bash
  cd twc-test-web
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
