TWC Innovations Contact Poratal Client-side application
## Routes
Welcome - /
<br>
Login - /login
<br>
Register - /register
<br>
Contacts - /contacts
<br>

## Tech Stack

ReactJS, Vite, TailwindCSS
## Run Locally


```bash
  https://github.com/ashandimantha2000/twc-test-web.git
```

Go to the project directory

```bash
  cd twc-test-web
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
